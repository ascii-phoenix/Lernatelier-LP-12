# Lernatelier-LP-12
## Beschreibung
Word/Outlook Add-in, das mit Ollama (z. B. Qwen2.5:3b) kommuniziert über .NET API Backend.

## Features
- Taskpane Add-in für Word & Outlook  
- Textanfragen an Ollama API  
- Antworten direkt im Add-in anzeigen
